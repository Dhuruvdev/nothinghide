About NothingHide

NothingHide is a breach exposure intelligence tool that helps you check if your email or password has been compromised in data breaches. All checks use lawful, publicly available sources only.

Email Breach Check

Queries 6+ public breach databases in parallel to find if your email appears in known data breaches. Sources include:

  • LeakCheck (7B+ records)
  • HackCheck
  • XposedOrNot
  • XposedOrNot Analytics
  • EmailRep (reputation)
  • DeXpose

Password Check

Uses the Have I Been Pwned Pwned Passwords API with k-anonymity protocol. Your password is NEVER transmitted to any server. Here's how it works:

  • Your password is hashed using SHA-1
  • Only the first 5 characters of the hash are sent to the API
  • The API returns all hash suffixes that match
  • Full comparison happens locally on your device

Full Scan

Combines email breach check and password check to provide a complete identity exposure analysis with risk assessment and actionable recommendations.

Privacy & Security

  • No data is ever stored or logged
  • Password uses k-anonymity - only partial hash transmitted
  • All sources are publicly available and lawful
  • Open source and transparent
